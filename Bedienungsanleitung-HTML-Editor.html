Bedienungsanleitung
HTML Editor

Einfache Anleitung zum Bearbeiten von Webseiten – kein Fachwissen nötig
Erstellt mit Cortex AI Solutions & Claude Sonnet 4.6 – Mai 2026

1. Programm starten

Der HTML Editor ist eine einzige Datei: HTML-Editor.html
Sie müssen nichts installieren. So starten Sie das Programm:

1
Speichern Sie die Datei HTML-Editor.html auf Ihrem Computer, z. B. auf dem Desktop.
2
Machen Sie einen Doppelklick auf die Datei. Sie öffnet sich automatisch im Browser (Firefox, Chrome oder Edge).
3
Der Editor erscheint mit einer blauen Titelleiste. Er ist bereit zur Benutzung.
💡 Tipp: Klicken Sie mit der rechten Maustaste auf die Datei und wählen Sie „Öffnen mit" → „Firefox" oder „Chrome", falls sie sich nicht automatisch öffnet.

2. Arbeitspfad festlegen

Der Arbeitspfad ist der Ordner auf Ihrem Computer, in dem Ihre Website gespeichert ist – also der Ordner mit Ihren HTML-Dateien, Bildern und dem Design (CSS).

⚠️ Wichtig: Legen Sie den Arbeitspfad immer zuerst fest, bevor Sie eine Datei öffnen! Nur so werden Bilder und das Design korrekt in der Vorschau angezeigt.
1
Klicken Sie oben links auf den Button 📁 Arbeitspfad festlegen.
2
Es öffnet sich ein Ordner-Auswahl-Fenster. Navigieren Sie zum Ordner Ihrer Website und klicken Sie auf „Ordner auswählen".
3
Der Editor liest den gesamten Ordner ein – alle Bilder und CSS-Dateien werden geladen. In der Info-Leiste unterhalb der Werkzeugleiste sehen Sie den Namen des gewählten Ordners.
💡 Was passiert dabei? Der Editor liest alle Dateien Ihres Website-Ordners ein. Das Design Ihrer Seite (CSS) wird automatisch eingebettet, sodass die Vorschau genauso aussieht wie die echte Website. Ihre Bilder werden automatisch angezeigt.
💡 Direktes Speichern: Weil der Editor jetzt weiß, wo Ihr Website-Ordner liegt, kann er beim Speichern direkt in diesem Ordner schreiben – ohne Downloads-Umweg. Details dazu in Abschnitt 12 „Datei speichern".

3. HTML-Datei öffnen

Nachdem der Arbeitspfad festgelegt wurde, öffnen Sie die Seite, die Sie bearbeiten möchten:

1
Klicken Sie auf den Button 📂 Öffnen.
2
Es öffnet sich ein Datei-Auswahl-Fenster. Suchen Sie Ihre HTML-Datei (z. B. termine.html) und klicken Sie auf „Öffnen".
3
Die Webseite erscheint als Vorschau. In der Info-Leiste wird der Dateiname angezeigt. Bilder und Design werden automatisch dargestellt – vorausgesetzt, der Arbeitspfad wurde zuvor festgelegt.
📌 Hinweis: Die Info-Leiste unterhalb der Werkzeugleiste zeigt Ihnen immer, welcher Ordner (Arbeitspfad) und welche Datei gerade geöffnet ist.
💡 Warum das wichtig ist: Öffnen Sie die Datei über diesen Weg (statt z. B. per Doppelklick im Explorer), merkt sich der Editor genau, wo sie liegt. Das sorgt dafür, dass „💾 Speichern" später die richtige Datei direkt ersetzt, ohne Nachfrage – siehe Abschnitt 12.

4. Ansicht wechseln

Direkt unterhalb der Werkzeug-Leiste gibt es drei Schaltflächen:

Schaltfläche Was man sieht
👁 Vorschau Nur die Vorschau (Standard – am besten zum Bearbeiten geeignet)
⬛ Geteilt Links die Vorschau, rechts der HTML-Code – beides gleichzeitig
</> Code Nur den HTML-Quellcode (für Fortgeschrittene)
💡 Empfehlung: Arbeiten Sie in der Vorschau-Ansicht – so sehen Sie die Seite so, wie sie auf der Website erscheint, und haben die meiste Fläche zum Bearbeiten.

5. Elemente auswählen

Bevor Sie etwas bearbeiten, müssen Sie zuerst das gewünschte Element auswählen. Ein „Element" ist ein Bestandteil der Seite, z. B. ein Textabsatz, eine Überschrift, ein Bild oder ein Kasten.

1
Einmal klicken auf ein Element in der Vorschau.
2
Das Element bekommt einen blauen Rahmen – es ist jetzt ausgewählt. Der Editor wählt automatisch das nächstgelegene sinnvolle Element aus (z. B. den Absatz oder die Überschrift, nicht einzelne Wörter).
3
In der Statusleiste unten sehen Sie, welches Element ausgewählt ist. Jetzt können Sie die Schaltflächen in der Werkzeugleiste verwenden.
💡 Tipp: Wenn Sie ein sehr kleines Element nicht treffen können, klicken Sie auf ein größeres Element daneben und verschieben Sie es dann mit den Pfeilen „↑ Hoch" / „↓ Runter".

6. Text bearbeiten

Es gibt zwei Möglichkeiten, einen Text zu ändern:

Möglichkeit A – Doppelklick (einfachste Methode)

1
Machen Sie einen Doppelklick direkt auf den Text, den Sie ändern möchten.
2
Der Bereich bekommt einen grünen Rahmen – jetzt können Sie direkt tippen, löschen und ändern.
3
Drücken Sie Enter oder klicken Sie irgendwo außerhalb, um die Bearbeitung abzuschließen. Die Änderung wird sofort übernommen.

Möglichkeit B – über die Werkzeugleiste

1
Einmal klicken auf ein Element – es bekommt einen blauen Rahmen.
2
Klicken Sie oben auf ✏️ Text bearbeiten. Der grüne Bearbeitungsrahmen erscheint. Der Editor wechselt automatisch in den Text-Modus.
💡 Ganzen Bereich bearbeiten: Bei langen Texten (z. B. Gedichten) können Sie den gesamten umgebenden Bereich (z. B. ein Kasten-Element) einmal anklicken und dann auf „✏️ Text bearbeiten" klicken. So bearbeiten Sie alle Zeilen auf einmal, statt jede einzeln anzuklicken.
📌 Abbrechen ohne Speichern: Drücken Sie Esc, um die Bearbeitung abzubrechen – Ihre Änderungen werden dann verworfen und der ursprüngliche Text bleibt erhalten.

7. Elemente kopieren, verschieben, löschen

Wählen Sie zuerst ein Element aus (einmal klicken, blauer Rahmen), dann verwenden Sie diese Schaltflächen:

Schaltfläche Tastenkürzel Was passiert
📋 Kopieren Strg+C Element wird in die Zwischenablage kopiert
⧉ Duplizieren Strg+D Eine Kopie des Elements wird direkt darunter eingefügt
📌 Einfügen Strg+V Kopiertes Element wird nach dem ausgewählten eingefügt
↑ Hoch – Element eine Position nach oben schieben
↓ Runter – Element eine Position nach unten schieben
🗑 Löschen Entf Ausgewähltes Element löschen (mit Sicherheitsabfrage)
↩ Rückgängig Strg+Z Letzte Änderung rückgängig machen (bis zu 20 Schritte)
💡 Beispiel – Veranstaltungseintrag duplizieren: Klicken Sie auf einen bestehenden Eintrag, dann auf „⧉ Duplizieren". Die Kopie erscheint direkt darunter. Doppelklick auf den Text der Kopie, um ihn anzupassen.
🆕 Rückgängig: Mit ↩ Rückgängig (oder Strg+Z) können Sie jede Änderung Schritt für Schritt rückgängig machen – bis zu 20 Schritte in die Vergangenheit. Der Button ist ausgegraut, wenn es nichts zum Rückgängigmachen gibt.

8. Text-Modus und Foto-Modus

Der Editor hat zwei Arbeitsmodi, die Sie oben in der Werkzeugleiste umschalten können:

✎ Text-Modus (Standard)

  • Klick auf Element → blauer Rahmen (kein Cursor)
  • Doppelklick → Text direkt bearbeiten
  • Klick in leere Fläche → Auswahl aufheben
  • Keine blauen Positionslinien
  • Ideal für: Texte ändern, Elemente verschieben, Hyperlinks setzen

📷 Foto-Modus

  • Klick auf Element → blauer Rahmen + blaue Linie darunter
  • Klick in leere Fläche → blaue Linie zeigt Einfügepunkt
  • Foto geht genau dorthin, wo die blaue Linie erscheint
  • Ideal für: Fotos einfügen, Bilder positionieren
📌 Automatischer Wechsel: Wenn Sie auf „✏️ Text bearbeiten" klicken, wechselt der Editor automatisch in den Text-Modus. Wenn Sie auf „🖼 Foto einfügen" klicken oder ein Foto hineinziehen, wechselt er automatisch in den Foto-Modus. Sie können aber auch jederzeit manuell zwischen den Modi wechseln.

9. Foto einfügen

Es gibt zwei Möglichkeiten, ein Foto einzufügen. In beiden Fällen zeigt eine blaue Linie in der Vorschau genau an, wo das Foto landen wird.

Möglichkeit A – Drag & Drop (einfachste Methode)

1
Öffnen Sie den Windows-Datei-Explorer neben dem Browser-Fenster.
2
Ziehen Sie das gewünschte Foto-Bild direkt in die Vorschau des Editors. Während des Ziehens erscheint eine leuchtend blaue Linie, die zeigt, wo das Foto eingefügt wird.
3
Lassen Sie die Maustaste los – der Foto-Dialog öffnet sich mit dem Bild bereits vorgeladen und einer Vorschau.
4
Passen Sie Breite und Bildbeschreibung an und klicken Sie auf ✔ Einfügen.

Möglichkeit B – über die Werkzeugleiste

1
Aktivieren Sie den 📷 Foto-Modus (Button in der Werkzeugleiste).
2
Klicken Sie in der Vorschau auf die Stelle, nach der das Foto eingefügt werden soll. Eine blaue Linie erscheint und zeigt genau an, wohin das Foto kommt. Sie können auf ein beliebiges Element klicken oder in einen leeren Bereich zwischen zwei Elementen.
3
Klicken Sie auf 🖼 Foto einfügen. Ein Dialogfenster öffnet sich.
4
Klicken Sie auf „📷 Foto aus Dateimanager wählen …". Es öffnet sich der normale Datei-Auswahl-Dialog Ihres Computers. Navigieren Sie zum Bild-Ordner Ihrer Website und wählen Sie das gewünschte Bild aus. Nach der Auswahl erscheint eine kleine Vorschau im Dialog.
5
Wählen Sie die gewünschte Breite:
  • Volle Breite – Bild nimmt die volle Seitenbreite ein
  • Halbe Breite – Bild nimmt die halbe Breite ein
  • Klein – max. 300 Pixel
  • Mittel – max. 500 Pixel
  • Größe übernehmen – Größe des ausgewählten Elements (falls es ein Bild ist)
6
Geben Sie einen Alt-Text ein – eine kurze Beschreibung des Bildes, z. B. „Vereinstreffen im Saal". Das ist wichtig für blinde Nutzer und Suchmaschinen. Der Editor schlägt automatisch einen Text aus dem Dateinamen vor.
7
Klicken Sie auf ✔ Einfügen. Das Bild erscheint in der Vorschau genau an der durch die blaue Linie markierten Stelle.
💡 Automatischer Pfad: Wenn der Arbeitspfad festgelegt ist und das Bild aus dem Website-Ordner stammt, berechnet der Editor den korrekten relativen Pfad vollautomatisch – der grüne Pfad im Dialog bestätigt das. Stammt das Bild von einem anderen Ort, erscheint eine gelbe Warnung; der Pfad sollte dann manuell geprüft werden.
📌 Foto ans Ende: Mit 🖼 Foto ans Ende fügen Sie ein Bild immer ganz am Ende der Seite ein – unabhängig vom Modus und ohne vorherige Auswahl.

10. Foto ersetzen

Wenn Sie ein bereits eingefügtes Bild durch ein anderes ersetzen möchten, müssen Sie es nicht löschen und neu einfügen.

1
Klicken Sie auf das Bild, das Sie ersetzen möchten (blauer Rahmen erscheint).
2
Der Button 🖼 Foto ersetzen in der Werkzeugleiste wird jetzt aktiv (er ist sonst ausgegraut). Klicken Sie darauf.
3
Der Foto-Dialog öffnet sich. Klicken Sie auf „📷 Foto aus Dateimanager wählen …", wählen Sie das neue Bild aus dem Datei-Dialog Ihres Computers, und passen Sie Größe und Alt-Text an.
4
Klicken Sie auf Ersetzen. Das alte Bild wird direkt durch das neue ersetzt – kein doppeltes Bild, keine Verschiebung.
📌 Hinweis: Der Button „Foto ersetzen" ist nur aktiv, wenn ein Bild (<img>) ausgewählt ist. Bei allen anderen Elementen ist er ausgegraut.

11. Hyperlink bearbeiten

Mit dem Hyperlink-Button können Sie einem Element einen Link hinzufügen – z. B. zu einer anderen Seite Ihrer Website oder zu einer externen Webseite.

1
Klicken Sie auf das Element, das verlinkt werden soll (blauer Rahmen).
2
Klicken Sie auf 🔗 Hyperlink. Ein Dialogfenster öffnet sich.
3
Wählen Sie den Link-Typ:
  • 📄 Seite im Arbeitspfad: Klicken Sie auf „📄 Seite aus Dateimanager wählen …" und wählen Sie die gewünschte HTML-Datei aus Ihrem Website-Ordner. Der korrekte relative Pfad wird automatisch berechnet und grün angezeigt.
  • 🌐 Eigene URL: Geben Sie eine vollständige Webadresse ein, z. B. https://www.example.de. Falls Sie das „https://" vergessen, wird es automatisch ergänzt.
4
Optional: Aktivieren Sie die Checkbox „In neuem Tab öffnen", wenn der Link in einem neuen Browser-Tab geöffnet werden soll (empfohlen für externe Links).
5
Klicken Sie auf ✔ Übernehmen.
💡 Funktioniert für alle Elemente: Sie können nicht nur Text-Links, sondern auch Bilder, Kacheln oder Überschriften verlinken. Wenn das ausgewählte Element selbst kein Link ist, wird es automatisch in einen Link eingebettet. Wenn es bereits ein Link ist, wird der bestehende Link bearbeitet.

12. Datei speichern

Wie „💾 Speichern" sich verhält, hängt davon ab, wie Sie die Datei geöffnet haben:

Fall A: Sie haben die Datei über „📁 Arbeitspfad festlegen" + „📂 Öffnen" geladen (Normalfall)

1
Klicken Sie oben auf 💾 Speichern – oder drücken Sie Strg+S.
2
Der Editor weiß, in welchem Ordner die Datei liegt (Sie haben ihn ja als Arbeitspfad festgelegt) und ersetzt die Original-Datei direkt – ohne Nachfrage, ohne Fenster, ohne Downloads-Ordner. Es entstehen keine zusätzlichen Kopien oder Versionsnummern.
💡 Das ist das empfohlene Vorgehen: Arbeitspfad festlegen → Datei darüber öffnen → bearbeiten → Speichern. So landet jede Änderung garantiert in der richtigen Datei am richtigen Ort.

Fall B: Sie haben die Datei ohne vorherigen Arbeitspfad geöffnet (z. B. per Doppelklick oder aus einem anderen Ordner)

1
Klicken Sie auf 💾 Speichern. Da der Editor den Speicherort nicht kennt, öffnet sich das Windows-Speicherfenster.
2
Wechseln Sie dort in den Ordner Ihrer Website, in dem die Original-Datei liegt, und klicken Sie auf Speichern. Windows fragt: „Datei existiert bereits – ersetzen?" – antworten Sie mit Ja.
3
Bei jedem weiteren Speichern dieser Datei merkt sich der Editor den gewählten Ort – ein Klick auf „💾 Speichern" genügt dann, es öffnet sich kein Fenster mehr.
💡 Originale Pfade bleiben erhalten: Die gespeicherte Datei enthält die gleichen Verweise auf CSS-Dateien und Bilder wie vorher – nur mit Ihren inhaltlichen Änderungen. Das eingebettete Design (CSS) der Vorschau wird nicht in die gespeicherte Datei geschrieben.
📌 Tipp bei Fehlern: Nutzen Sie ↩ Rückgängig (Strg+Z), um Änderungen Schritt für Schritt zurückzunehmen, bevor Sie speichern. Solange Sie nicht auf „💾 Speichern" geklickt haben, können Sie die Originaldatei auch einfach erneut öffnen – alle Ihre Änderungen gehen dann verloren, aber die Originaldatei ist unverändert.
📌 Hinweis für ältere Browser (z. B. Firefox): Falls sich kein Speicherfenster öffnet, lädt der Browser die Datei stattdessen in den Downloads-Ordner herunter – dort ggf. mit einer Nummerierung wie „(1)" versehen, wenn dort schon eine Datei mit diesem Namen liegt. Kopieren Sie sie dann von dort in den Ordner Ihrer Website und ersetzen Sie die alte Datei.

13. Neue Unterseite anlegen (z. B. neue Pressemitteilung)

Der einfachste Weg zu einer neuen Unterseite: Sie nehmen eine bestehende, ähnliche Seite als Vorlage, speichern sie unter einem neuen Namen und ändern dann den Inhalt. So entsteht z. B. aus der letzten Pressemitteilung eine neue – mit dem gleichen Aussehen, aber neuem Text und neuen Fotos.

Schritt A: Neue Seite aus einer Vorlage erstellen

1
Öffnen Sie die bestehende Seite, die der neuen am ähnlichsten ist (z. B. die letzte Pressemitteilung pm_2026_03_literaturwerkstatt.html).
2
Klicken Sie sofort auf 💾 Speichern unter … und geben Sie der Datei einen neuen Namen (z. B. pm_2026_07_scheunenlesung.html). Wichtig: Speichern Sie die neue Datei im selben Ordner, in dem auch die Vorlage liegt – nur dann stimmen alle Verweise auf Bilder und Design.
3
Ändern Sie jetzt in Ruhe die Inhalte: Überschrift und Texte per Doppelklick bearbeiten, Fotos mit 🔄 Foto ersetzen austauschen. Jedes „💾 Speichern" landet nun automatisch in der neuen Datei – die Vorlage bleibt unverändert.

Schritt B: Die neue Seite auf der Übersichtsseite verlinken

4
Öffnen Sie die Übersichtsseite (z. B. pressemitteilungen.html) und wählen Sie den neuesten vorhandenen Eintrag aus (blauer Rahmen). Klicken Sie auf ⧉ Duplizieren – eine Kopie des Eintrags erscheint direkt darunter.
5
Bearbeiten Sie in der Kopie Datum, Überschrift und Ankündigungstext per Doppelklick. Mit ↑ Hoch schieben Sie den neuen Eintrag an die oberste Stelle.
6
Ganz wichtig: Klicken Sie im neuen Eintrag auf den „Weiterlesen"-Knopf (er bekommt einen blauen Rahmen) und dann oben auf 🔗 Hyperlink. Wählen Sie über „📄 Seite aus Dateimanager wählen …" Ihre neue Seite aus Schritt A aus und klicken Sie auf ✔ Übernehmen. Sonst führt der Knopf noch zur alten Seite, von der Sie kopiert haben!
7
Speichern Sie die Übersichtsseite mit 💾 Speichern. Fertig!
💡 Merkhilfe: Eine neue Pressemitteilung besteht immer aus zwei Dingen: der neuen Unterseite mit dem vollständigen Text (Schritt A) und dem Eintrag auf der Übersichtsseite, dessen „Weiterlesen"-Knopf auf die neue Unterseite zeigt (Schritt B). Wenn „Vollständiger Text" auf eine falsche Seite führt, ist fast immer der Link aus Schritt 6 noch nicht angepasst.

15. Häufige Fragen

Die Bilder werden in der Vorschau nicht angezeigt.

Der Arbeitspfad wurde wahrscheinlich nicht festgelegt. Gehen Sie so vor: (1) Klicken Sie auf „📁 Arbeitspfad festlegen" und wählen Sie den Hauptordner Ihrer Website. (2) Klicken Sie dann auf „📂 Öffnen" und laden Sie die HTML-Datei neu. Hinweis: Das Foto-Einfügen-Dialogfenster zeigt eine gelbe Warnung, wenn ein gewähltes Bild nicht im Arbeitspfad gefunden wird – in diesem Fall den Arbeitspfad neu laden und erneut versuchen.

Das Design (CSS) sieht in der Vorschau falsch aus.

Auch hier hilft es, den Arbeitspfad zuerst festzulegen. Der Editor sucht im Website-Ordner nach CSS-Dateien und bettet sie in die Vorschau ein. Stellen Sie sicher, dass der Arbeitspfad auf den Hauptordner der Website zeigt, in dem auch die CSS-Dateien liegen.

Der Button „Foto ersetzen" ist ausgegraut.

Der Button ist nur aktiv, wenn ein Bild (<img>) ausgewählt ist. Klicken Sie zuerst auf das Bild in der Vorschau – es muss den blauen Auswahlrahmen bekommen. Danach wird der Button aktiv. Stellen Sie außerdem sicher, dass Sie sich im Foto-Modus befinden oder klicken Sie direkt auf „🖼 Foto ersetzen" – der Editor wechselt dann automatisch.

Wozu brauche ich Text-Modus und Foto-Modus?

Die zwei Modi verhindern, dass sich Textbearbeitung und Foto-Einfügen gegenseitig stören. Im Text-Modus sehen Sie keine ablenkenden blauen Positionslinien – ideal zum ruhigen Arbeiten an Texten. Im Foto-Modus zeigt die blaue Linie jederzeit genau an, wo das nächste Foto landen wird. Der Editor wechselt automatisch in den richtigen Modus, wenn Sie auf die entsprechenden Buttons klicken.

Ich kann keinen Hyperlink auf ein Bild setzen – geht das?

Ja, das geht. Klicken Sie auf das Bild (blauer Rahmen), dann auf „🔗 Hyperlink". Das Bild wird automatisch in einen anklickbaren Link eingebettet.

Der Link öffnet sich nicht auf der echten Website.

Prüfen Sie, ob der Link-Pfad stimmt. Bei internen Links (Seiten im Arbeitspfad) berechnet der Editor den Pfad automatisch. Bei externen Links prüfen Sie, ob die Adresse vollständig ist (z. B. https://www.example.de). Vergewissern Sie sich außerdem, dass die verlinkte Datei tatsächlich im richtigen Ordner auf dem Server liegt.

Ich habe versehentlich etwas gelöscht oder falsch bearbeitet.

Klicken Sie auf ↩ Rückgängig in der Werkzeugleiste – oder drücken Sie Strg+Z. Sie können bis zu 20 Schritte zurückgehen. Solange Sie nicht auf „💾 Speichern" geklickt haben, können Sie die Originaldatei auch einfach erneut öffnen – alle Änderungen gehen dann verloren, aber die Originaldatei ist unverändert.

Umlaute (ä, ö, ü, ß) werden komisch angezeigt.

Stellen Sie sicher, dass die HTML-Datei oben im Code den Eintrag <meta charset="UTF-8"> enthält. Der Editor speichert immer in UTF-8, was die korrekte Einstellung ist.

Meine Änderungen sind nach dem Speichern nicht auf der Website zu sehen.

Prüfen Sie zuerst, ob Sie beim Speichern wirklich die Datei im Webseiten-Ordner ersetzt haben (und nicht versehentlich an einem anderen Ort gespeichert). Danach muss die geänderte Datei wie gewohnt auf den Server hochgeladen werden. Leeren Sie anschließend den Browser-Cache, falls die Seite noch das alte Aussehen zeigt (Strg+Shift+R).

„Speichern" legt eine zusätzliche Datei mit Versionsnummer an, statt das Original zu ersetzen.

Das passiert, wenn die Datei nicht über „📁 Arbeitspfad festlegen" + „📂 Öffnen" geladen wurde (z. B. per Doppelklick im Explorer geöffnet, oder Ihr Browser unterstützt das direkte Speichern nicht). In diesem Fall speichert der Editor über den Downloads-Ordner, und Windows hängt bei bereits vorhandenem Dateinamen automatisch eine Nummer an. Abhilfe: Legen Sie zuerst den Arbeitspfad fest und öffnen Sie die Datei darüber (Abschnitt 2 und 3) – dann ersetzt „Speichern" die Original-Datei direkt, ohne Umweg über Downloads. Räumen Sie überzählige Kopien im Downloads-Ordner danach von Hand auf.

Der „Weiterlesen"-Knopf meines neuen Eintrags führt zur falschen Seite.

Das passiert, wenn ein Eintrag von einem älteren kopiert wurde – der Knopf zeigt dann noch auf die alte Seite. Klicken Sie auf den „Weiterlesen"-Knopf (blauer Rahmen), dann auf 🔗 Hyperlink, und wählen Sie die richtige Zielseite aus. Die Schritt-für-Schritt-Anleitung steht in Abschnitt 13 – Neue Unterseite anlegen.

Mein Foto ist an der falschen Stelle gelandet (z. B. neben einer Überschrift).

Das Foto wird immer nach dem blau umrahmten Element eingefügt. Wenn eine Überschrift ausgewählt war, landet das Foto direkt unter der Überschrift. Das Dialogfenster zeigt die Einfügeposition vor dem Einfügen an – bei einer Überschrift erscheint zusätzlich eine gelbe Warnung. Zum Korrigieren: ↩ Rückgängig klicken, dann den Textabsatz des Beitrags anklicken (nicht die Überschrift) und das Foto erneut einfügen.
